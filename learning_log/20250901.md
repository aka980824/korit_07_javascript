# 자리 이동후

git clone 방법

1. git clone 링크
2. shift + 우클릭 -> . code
3. 폴더 경로 확인
4. 자바스크립드폴더 기준으로 시프드 우클릭 + code
  1. 이유 0> 깃이이미 잇음
  2. 푸쉬한것처럼 깃 인잇부터 시작할 필요
  3. git config user.name ""
  4. git config user.email
